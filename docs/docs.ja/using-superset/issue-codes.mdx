# 問題コードのリファレンス

このページには、Superset に表示される可能性のある問題コードがリストされ、追加のコンテキストが提供されます。

## Issue 1000

```
The datasource is too large to query.
```

データソースが大きくなりすぎて現在のクエリを実行できず、タイムアウトになっている可能性があります。
データソースのサイズを小さくするか、データのサブセットのみを処理するようにクエリを変更することで、この問題を解決できます。

## Issue 1001

```
The database is under an unusual load.
```

データベースエンジンの負荷が異常に高くなったため、クエリがタイムアウトした可能性があります。
クエリを簡素化するか、データベースの負荷が軽減されるまで待ってから再試行してください。

## Issue 1002

```
The database returned an unexpected error.
```

データベースでエラーが発生したため、クエリが失敗しました。
これは、構文エラー、クエリのバグ、またはデータベース内のその他の内部エラーが原因である可能性があります。
これは通常、Superset 内の問題ではなく、クエリを処理する基盤となるデータベースの問題です。

## Issue 1003

```
There is a syntax error in the SQL query. Perhaps there was a misspelling or a typo.
```

クエリの構文エラーのため、クエリが失敗しました。
クエリ内で参照されているすべての列またはテーブルが存在し、スペルが正しいことを確認してください。

## Issue 1004

```
The column was deleted or renamed in the database.
```

クエリは、基になるデータソースに存在しない列を参照しているため失敗しました。
置換列を参照するようにクエリを変更するか、クエリからこの列を削除する必要があります。

## Issue 1005

```
The table was deleted or renamed in the database.
```

クエリは、基になるデータベースに存在しないテーブルを参照しているため失敗しました。
正しいテーブルを参照するようにクエリを変更する必要があります。

## Issue 1006

```
One or more parameters specified in the query are missing.
```

1つ以上のパラメータが不足しているため、クエリはデータベースに送信されませんでした。
クエリで参照されているすべてのパラメータを有効なJSONドキュメントで定義する必要があります。
パラメータのスペルが正しく、ドキュメントの構文が有効であることを確認してください。

## Issue 1007

```
The hostname provided can't be resolved.
```

新しいデータベースを追加する際に指定したホスト名が無効であり、解決できません。
ホスト名に誤字がないことを確認してください。

## Issue 1008

```
The port is closed.
```

新しいデータベースの追加時に指定したポートが開いていません。
ポート番号が正しいこと、およびデータベースがそのポートで実行され、リッスンしていることを確認してください。

## Issue 1009

```
The host might be down, and cannot be reached on the provided port.
```

新しいデータベースを追加する際に指定したホストが起動していないようです。
また、指定したポートでアクセスできません。
ホストへのアクセスをブロックするファイアウォールルールがないことを確認してください。

## Issue 1010

```
Superset encountered an error while running a command.
```

予期しない問題が発生し、Superset でコマンドの実行中にエラーが発生しました。
管理者に問い合わせてください。

## Issue 1011

```
Superset encountered an unexpected error.
```

Superset バックエンドで予期しない問題が発生しました。
管理者にご連絡ください。

## Issue 1012

```
The username provided when connecting to a database is not valid.
```

ユーザーがデータベースに存在しないユーザー名を入力しました。
ユーザー名が正しく入力され、データベースに存在することを確認してください。

## Issue 1013

```
The password provided when connecting to a database is not valid.
```

ユーザーが入力したパスワードが間違っています。
パスワードが正しく入力されているか確認してください。

## Issue 1014

```
Either the username or the password used are incorrect.
```

入力されたユーザー名が存在しないか、パスワードが間違っています。
ユーザー名とパスワードが正しく入力されていることを確認してください。

## Issue 1015

```
Either the database is spelled incorrectly or does not exist.
```

データベースの記述が間違っているか、データベースが存在しません。
正しく入力されているか確認してください。

## Issue 1016

```
The schema was deleted or renamed in the database.
```

スキーマが削除されたか、名前が変更されました。
スキーマが正しく入力され、存在していることを確認してください。

## Issue 1017

```
The user doesn't have the proper permissions to connect to the database
```

データベースに接続できませんでした。
サービスアカウントにプロジェクトの閲覧者ロールとジョブユーザーロールが付与されていることを確認してください。

## Issue 1018

```
One or more parameters needed to configure a database are missing.
```

データベースのテスト、作成、編集に必要なパラメータがすべて揃っていません。
必要なパラメータとそのパラメータが揃っていることを再度ご確認ください。

## Issue 1019

```
The submitted payload has the incorrect format.
```

リクエストペイロードの形式が正しいことを確認してください (例: JSON)。

## Issue 1020

```
The submitted payload has the incorrect schema.
```

リクエストペイロードに予期されたスキーマがあることを確認してください。

## Issue 1021

```
Results backend needed for asynchronous queries is not configured.
```

Superset のインスタンスに、非同期クエリに必要な結果バックエンドが設定されていません。
詳細については、管理者にお問い合わせください。
## Issue 1022

```
Database does not allow data manipulation.
```

このデータベースに対しては `SELECT` ステートメントのみが許可されています。
このデータベースで DML (データ操作言語) を実行する必要がある場合は、管理者に問い合わせてください。

## Issue 1023

```
CTAS (create table as select) doesn't have a SELECT statement at the end.
```

CTAS (create table as select) として実行されるクエリの最後のステートメントは、SELECT ステートメントである必要があります。
クエリの最後のステートメントが SELECT であることを確認してください。

## Issue 1024

```
CVAS (create view as select) query has more than one statement.
```

CVAS（create view as select）を実行する場合、クエリには単一のステートメントが必要です。
クエリに単一のステートメントが含まれており、最後のセミコロン以外に余分なセミコロンがないことを確認してください。

## Issue 1025

```
CVAS (create view as select) query is not a SELECT statement.
```

CVAS（create view as select）を実行する場合、クエリはSELECT文である必要があります。
クエリに単一の文があり、それがSELECT文であることを確認してください。

## Issue 1026

```
Query is too complex and takes too long to run.
```

送信されたクエリは複雑すぎるため、Superset管理者が定義した時間制限内で実行できない可能性があります。
クエリを再確認し、最適化できるかどうかを確認してください。
または、管理者に連絡してタイムアウト期間を延長してください。

## Issue 1027

```
The database is currently running too many queries.
```

データベースの負荷が高く、クエリの実行数が多すぎる可能性があります。
しばらくしてからもう一度お試しいただくか、管理者にご連絡ください。

## Issue 1028

```
One or more parameters specified in the query are malformed.
```

クエリに不正なテンプレートパラメータが1つ以上含まれています。
クエリを確認し、すべてのテンプレートパラメータが二重中括弧で囲まれていることを確認してください（例："\{\{ ds \}\}"）。
その後、クエリを再度実行してください。

## Issue 1029

```
The object does not exist in this database.
```

スキーマ、列、またはテーブルのいずれかがデータベースに存在しません。

## Issue 1030

```
The query potentially has a syntax error.
```

クエリに構文エラーがある可能性があります。
確認して再度実行してください。

## Issue 1031

```
The results backend no longer has the data from the query.
```

クエリの結果は、一定期間後に結果バックエンドから削除されている可能性があります。
クエリを再実行してください。

## Issue 1032

```
The query associated with the results was deleted.
```

保存された結果に関連付けられたクエリは存在しません。
クエリを再実行してください。

## Issue 1033

```
The results stored in the backend were stored in a different format, and no longer can be deserialized.
```

クエリ結果はサポートされていない形式で保存されました。
クエリを再実行してください。

## Issue 1034

```
The database port provided is invalid.
```

指定されたデータベース ポートが 0 から 65535 までの整数であることを確認してください。

## Issue 1035

```
Failed to start remote query on a worker.
```

クエリは非同期ワーカーによって開始されませんでした。
詳細については、管理者にお問い合わせください。

## Issue 1036

```
The database was deleted.
```

参照されたデータベースが存在しないため、操作は失敗しました。
詳細については、管理者にお問い合わせください。
