# 国別地図の視覚化

国別マップのビジュアライゼーションでは、国を州、都道府県、その他の区分ごとに軽量なコロプレスマップとして作成できます。
サードパーティの地図サービスには依存しませんが、国の最上位区分の[ISO-3166-2](https://en.wikipedia.org/wiki/ISO_3166-2)コードをご提供いただく必要があります。
州や県の正式名称と比較すると、ISOコードは曖昧さが少なく、世界中のすべての地域で一意です。

## 含まれるマップ

現在の国リストは、ソース [legacy-plugin-chart-country-map/src/countries.ts](https://github.com/apache/superset/blob/master/superset-frontend/plugins/legacy-plugin-chart-country-map/src/countries.ts) に記載されています。

Country Maps ビジュアライゼーションには、以下の国の地図が既に含まれています。

<ul style={{columns: 3}}>
{countriesData.countries.map((country, index) => (
  <li key={index}>{country}</li>
))}
</ul>

## 新しい国を追加する

リストに新しい国を追加するには、[@superset-ui/legacy-plugin-chart-country-map](https://github.com/apache/superset/tree/master/superset-frontend/plugins/legacy-plugin-chart-country-map) 内のファイルを編集する必要があります。

1. [こちらの Jupyter Notebook](https://github.com/apache/superset/blob/master/superset-frontend/plugins/legacy-plugin-chart-country-map/scripts/Country%20Map%20GeoJSON%20Generator.ipynb) のガイドに従って、対象の国用の新しい GeoJSON ファイルを生成します。
2. [legacy-plugin-chart-country-map/src/countries.ts](https://github.com/apache/superset/blob/master/superset-frontend/plugins/legacy-plugin-chart-country-map/src/countries.ts) の国リストを編集します。
3. superset-frontend の依存関係をインストールします: `cd superset-frontend && npm install`
4. Superset プラグインのストーリーブックで国を確認します: `npm run plugins:storybook`
5. ソースコードから Superset をビルドしてインストールします。
